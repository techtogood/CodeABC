## codeABC
codeABC是一个少儿编程启蒙项目，包括6个编程主题，每个主题包括超过20个关卡。typescript编写，基于Phaser游戏引擎（https://github.com/photonstorm/phaser） ,不错的前端学习项目。目前开放了第一个主体的资源，如项目stars超过1k，将开发全部资源。

### 安装与启动

项目根目录下打开命令行工具

安装依赖

```shell
yarn
```

运行项目

```shell
yarn dev
```

打包项目

```shell
yarn build
```
