

## 橙狮编程

### 技术选型

| 技术       | 说明                 | 相关文档                                |
| ---------- | -------------------- | --------------------------------------- |
| Phaser 3   | h5游戏框架           | http://phaser.io/phaser3                |
| typescript | JavaScript的严格超集 | https://www.typescriptlang.org/         |
| webpack    | 前端打包工具         | https://webpack.docschina.org/concepts/ |

### 运行环境

| 环境    | 版本   |
| ------- | ------ |
| node.js | 14.6.0 |

### 安装与启动

项目根目录下打开命令行工具

安装依赖

```shell
yarn
```

运行项目

```shell
yarn dev
```

打包项目

```shell
yarn build
```
